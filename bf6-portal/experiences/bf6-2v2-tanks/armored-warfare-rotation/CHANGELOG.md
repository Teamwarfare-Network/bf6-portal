# Changelog

All notable changes to the Armored Warfare - 3 Map Rotation experience.

## [0.2.0] - 2025-12-02

### Added
- Vehicle registration system with debug messages
- Ammo crate resupply on main base entry/exit (2 crates per sortie)
- Abandonment detection system
- Global kill tracking for scoring

### Changed
- Published experience codes now live
- Finalized playtesting guide v0.2

### Known Issues
- Vehicle registration intermittent on first entry (workaround: enter vehicle twice)
- Cannot edit vehicle configuration on entry (Portal limitation)
- EOD Bot available as gadget due to spawn requirements (should not be used)

## [0.1.0] - 2025-12-01

### Added
- Initial experience creation
- Defense Nexus map support (up to 6v6)
- Manhattan Bridge map support (up to 4v4)
- Mirak Valley map support (up to 8v8)
- Basic scoring: first to 11 kills
- 15 minute time limit per round
- Engineer-only class restriction
- Main base spawn zones
