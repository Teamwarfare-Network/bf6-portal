# BF6 Core Reference Index

## Purpose

Store canonical BF6 portal facts shared by multiple skills.

## Generated API References

The following folders are scraped from BF Portal docs:

- `mod/` -> functions, enumerations, type aliases, variables, namespaces
- `modlib/` -> functions, classes

Indexes:

- `00-api-reference-index.md`
- `mod/00-api-index.md`
- `modlib/00-api-index.md`

Regenerate:

- `python bf6-portal/dev/reference_bf6_core/scripts/scrape_bfportal_function_refs.py`
